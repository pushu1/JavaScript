Logical Operators Three Type Kai Hote Hai

Operator Name
&& Logical AND
|| Logical OR

Logical AND (&&) Operator

Agar AND Operator Mai Dono Condition True Hogi To Hi Isme Statement Run Hota Hai Ak Bhi Value False Hone Par Yai Run Nahi Hoga.



Logical OR (||) Operator

Agar Two Condition Mai Sai Ak Bhi Condition True Hogi To Yai Operator Run Ho Jayega.



Logical Not (!) Operator

Isme Agar Condition True Hogi To False Show Karega Aur Agar Condition False Hogi To True Show Karega Isme Iska Ulta Hota Hai.
